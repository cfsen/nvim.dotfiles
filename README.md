# nvim dotfiles
This is my [neovim](https://neovim.io/) config, which was loosely based off using and modifying [kickstart.nvim](https://github.com/nvim-lua/kickstart.nvim). After rewriting it from scratch when moving to nvim 0.12, I decided to move it to it's own repo.

## Features
- Lightweight
- Uses neovim's native LSP and plugin manager.
- Set up for Rust, C#, Go, Python, and TS/web.
- Convenient rebinding of norwegian keys.
- The essential plugins for productivity (Telescope, treesitter, blink, etc.)
